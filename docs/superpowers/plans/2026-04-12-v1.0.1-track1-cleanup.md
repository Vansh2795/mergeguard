### Task 1: MCP server — repo validation + rate limiting (M2, M3)

**Files:**
- Modify: `src/mergeguard/mcp/server.py`

**M2 fix:** Add repo format validation to all MCP tool handlers. The CLI uses `r"^[\w.-]+/[\w.-]+$"` — use the same pattern:

```python
import re

_REPO_PATTERN = re.compile(r"^[\w.-]+/[\w.-]+$")

def _validate_repo(repo: str) -> None:
    if not _REPO_PATTERN.match(repo):
        raise ValueError(f"Invalid repo format: {repo!r}. Expected 'owner/repo'.")
```

Call `_validate_repo(repo)` at the start of each tool handler (`check_conflicts`, `get_risk_score`, `suggest_merge_order`).

**M3 fix:** Add a simple in-memory rate limiter per MCP session:

```python
import time

_last_call: dict[str, float] = {}
_MIN_INTERVAL = 2.0  # seconds between calls per tool

def _rate_check(tool_name: str) -> None:
    now = time.monotonic()
    last = _last_call.get(tool_name, 0.0)
    if now - last < _MIN_INTERVAL:
        raise ValueError(f"Rate limited: wait {_MIN_INTERVAL - (now - last):.1f}s")
    _last_call[tool_name] = now
```

Call `_rate_check(tool_name)` at the start of each tool handler.

### Task 3: Merge group regex — tighten pattern (M11)

**File:** `src/mergeguard/server/events.py:68-69`

Current:
```python
for match in re.finditer(r"#(\d+)", message):
    pr_numbers.add(int(match.group(1)))
```

Change to a more specific pattern that matches GitHub merge commit messages:
```python
for match in re.finditer(r"(?:Merge pull request |PR |pull request )#(\d+)", message, re.IGNORECASE):
    pr_numbers.add(int(match.group(1)))
```

This avoids matching issue references like "Fixes #100" or markdown headings.

### Task 5: GitLab — cache diffs per MR (M19)

**File:** `src/mergeguard/integrations/gitlab_client.py`

`_get_all_diffs()` is called by both `get_pr_files()` and `get_pr_diff()`. If both are called for the same MR, diffs are fetched twice.

Add a simple dict cache:

```python
def __init__(self, ...):
    ...
    self._diff_cache: dict[int, list[dict]] = {}

def _get_all_diffs(self, mr_iid: int) -> list[dict]:
    if mr_iid in self._diff_cache:
        return self._diff_cache[mr_iid]
    # ... existing fetch logic ...
    self._diff_cache[mr_iid] = diffs
    return diffs
```
